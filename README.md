### Day 1 搭建开发环境
#### 项目结构

    awsome-python3-webapp/  <- 根目录
    +- backup/              <- 备份目录
    +- conf/                <- 配置文件
    +- dist/                <- 打包目录
    +- www/                 <- web目录，存放.py文件
    |   +- static/          <- 存放静态文件
    |   +- templates/       <- 存放模板文件
    +- ios/                 <- 存放iOS App工程
    +- LICENSE              <- 代码LISENSE
